# Code of Conduct

This project follows the
[Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/),
version 2.1.

In short: be respectful, assume good intent, and keep discussion focused on the
project. Harassment and personal attacks are not tolerated.

## Reporting

If you experience or witness unacceptable behavior, report it to the
maintainers by opening a confidential issue or contacting the project owner
listed in the repository. Reports will be reviewed and handled in good faith.
